# TripGenie-AI

TripGenie-AI is an AI-powered web application that helps users plan trips by generating personalized travel itineraries.

## Features
- AI-generated trip itineraries
- User authentication
- Destination search
- Interactive maps
- Save and manage trips

## Tech Stack
- Next.js
- React
- JavaScript / TypeScript
- Tailwind CSS
- AI API
- Google Places API
- Mapbox
- Convex
- Clerk

## Setup
1. Clone the repository
2. Install dependencies using `npm install`
3. Add required environment variables
4. Run the project using `npm run dev`

## Usage
- Sign up or log in
- Enter destination details
- Generate a trip plan
- View itinerary and map

## Future Improvements
- Budget planning
- Trip sharing
- Hotel and flight recommendations
